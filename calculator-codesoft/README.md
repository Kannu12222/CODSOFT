# 🧮 Calculator - CODSOFT Internship

A simple and responsive Calculator web application developed using **HTML**, **CSS**, and **JavaScript** as part of the **CODSOFT Web Development Internship**.

## 🚀 Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Percentage (%) calculation
- Delete (DEL) functionality
- All Clear (AC) button
- Error handling for invalid expressions
- Responsive and clean user interface
- Built using CSS Grid layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
