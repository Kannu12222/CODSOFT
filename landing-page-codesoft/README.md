# 🚀 Responsive Landing Page

A modern and responsive landing page developed using **HTML** and **CSS** as part of the **CODSOFT Web Development Internship**. The project focuses on creating an attractive user interface with a clean layout and responsive design.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and clean user interface
- Hero section with call-to-action button
- Features section
- Services section
- Statistics section
- Customer testimonials
- Call-to-action section
- Footer with social media icons
- Smooth hover effects and transitions
- Font Awesome icons

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome
